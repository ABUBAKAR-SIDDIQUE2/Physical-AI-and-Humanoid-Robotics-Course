# Chapter Spec: The Robotic Nervous System (ROS 2)

**ID**: `02-ros2`
**Title**: The Robotic Nervous System (ROS 2)
**Folder**: `robotic-nervous-system-ros2`
**Learning Outcomes**:
- Understand the architecture of ROS 2 (Nodes, Topics, Services, Actions).
- Install and configure ROS 2 Humble/Iron on Ubuntu.
- Create a custom ROS 2 Python package.
- Write a Publisher and Subscriber node.
- Launch multiple nodes using launch files.

## Structure
1. **Overview**: ROS 2 as the middleware for robotics.
2. **Why This Matters**: Modular software design is essential for complex robots.
3. **Key Concepts**:
   - The Graph Metaphor (Nodes & Edges).
   - DDS (Data Distribution Service) middleware.
   - Topics (Pub/Sub), Services (Req/Res), Actions (Goal/Feedback).
4. **Detailed Explanations**:
   - Workspaces (`colcon build`).
   - Package structure (`setup.py`, `package.xml`).
   - The `rclpy` library.
5. **Hands-On Activity**:
   - Create `my_robot_controller` package.
   - Write `draw_circle.py` to control the Turtlesim turtle.
6. **Example**: Controlling a simulated servo motor via a topic.
7. **Troubleshooting**: Sourcing `setup.bash`, network discovery issues.
8. **Summary**: You now have a nervous system.
9. **Assessment**: Write a node that subscribes to a number and publishes its square.
